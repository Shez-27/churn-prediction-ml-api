# Churn Prediction ML API

This project implements an end-to-end machine learning pipeline to predict customer churn.
The solution includes data preprocessing, model training, evaluation, and deployment via a FastAPI REST API.

## Tech Stack
- Python
- scikit-learn
- XGBoost / CatBoost
- FastAPI
- Pandas, NumPy

## Project Structure
- notebooks/: EDA and experimentation
- src/: Training and preprocessing scripts
- app/: FastAPI application
- models/: Saved trained models
